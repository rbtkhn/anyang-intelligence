# Executive Memory

## Purpose

Executive memory is the durable context layer of the organization.

It preserves the facts, decisions, assumptions, commitments, risks, and open loops that leadership needs in order to act coherently.

## Memory Objects

- Organization profile.
- Strategy and goals.
- People and roles.
- Active projects.
- Decisions.
- Assumptions.
- Risks.
- Metrics.
- Customers and stakeholders.
- Follow-ups.
- Lessons learned.

## Operating Rule

The Executive OS should separate observed facts from interpretations and recommendations.

```text
Fact: What source evidence says.
Interpretation: What the system thinks it means.
Recommendation: What the system suggests doing next.
```

## Output

The memory layer should feed decision memos, weekly reviews, risk registers, and learning loops.
