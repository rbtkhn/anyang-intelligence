# Learning Loop

## Purpose

The learning loop compares expected outcomes with actual outcomes so the organization improves its model of reality.

## What It Tracks

- Decisions made.
- Assumptions behind each decision.
- Expected outcomes.
- Actual outcomes.
- Surprises.
- Lessons.
- Changed priorities.
- Updated risks.

## Review Questions

- What did we believe?
- What happened?
- What was misunderstood?
- What worked better than expected?
- What should change in the operating model?
- What should be remembered next time?

## Standard

The Executive OS should make learning visible without turning every review into blame. The goal is better judgment over time.
