# Learning Core Approval Exemplar

This worked example shows what meaningful parent approval looks like after a provisional plan draft is returned for review.

The case is fictionalized but follows the lane's real authority rules.

## Context

- a provisional first-month plan draft has been prepared
- the parent reviews the draft before any child-facing use
- one screen-time rule and one evidence-saving preference still need clarification

## Approval Receipt

```text
Approval Receipt

Lane: Learning Core
Workflow surface: parent plan approval
Date: 2026-07-08

Approval decision under review:
Whether a provisional first-month learning plan may be used.

Approver role:
Parent or guardian

Authority actually held:
- plan-use approval
- child-facing language approval
- privacy and save/share approval
- support-boundary judgment

What the approver reviewed:
- the provisional plan draft
- the visible assumptions
- the missing-input note
- the proposed evidence-saving approach

Evidence shown to the approver:
- parent intake summary
- provisional assumption list
- draft weekly rhythm
- proposed portfolio categories

Open uncertainties still visible at approval time:
- final screen-time preference
- whether short reflection notes may be saved weekly

Decision outcome:
- approved with changes

Changes required before use:
- update the screen-time rule
- narrow the evidence-saving rule to parent-approved categories only

What remains not approved:
- any child-facing use until the revised draft reflects those two changes

Meaningful-approval assessment:
- strong

Why the approval was meaningful or symbolic:
The parent held the right authority, reviewed the real boundary items, and changed two workflow-relevant rules before use.

Next required follow-up:
- revise the draft
- update the parent approval record
- return the revised version if any new child-facing wording appears
```

## Why This Approval Matters

This is not a symbolic thumbs-up. The approval changed the plan state, narrowed privacy rules, and preserved parent authority over actual use.
