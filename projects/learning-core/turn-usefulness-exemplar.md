# Learning Core Turn Exemplar

This worked example shows a governed intake-to-draft turn for Learning Core.

The case is fictionalized but shaped to match the lane's real authority and evidence rules.

## Input Context

Parent intake summary:

- learner is early elementary
- parent wants a calm first-month reading and math rhythm
- parent approves drafting from the intake response
- parent wants all child-facing language reviewed before use
- schedule is clear, but the portfolio evidence approach needs a small clarification

## Orchestration Steps Performed Inside The Turn

1. Check drafting approval, child-facing approval authority, and save/share boundaries.
2. Confirm minimum learner context, schedule reality, and starter resources.
3. Classify the intake as `Provisional` because the core safety boundary is clear but one evidence detail remains open.
4. Prepare the missing-input list and provisional assumptions before any draft is attempted.
5. Stop at parent review rather than treating a provisional draft as approved for use.

## Produced Artifacts Or Decisions

- intake classification: `Provisional`
- missing-input list
- provisional-assumption list
- safe next action: draft only with visible assumptions and return for parent review

## Approval Stop Point

Parent or guardian approval remains required for:

- child-facing language
- final plan use
- save/share decisions
- any shift from provisional assumptions to accepted learner-specific claims

## Why This Counts As A Small Workflow Turn

The turn did not only summarize the intake. It triaged authority, checked evidence boundaries, classified the case, prepared the next drafting state, and held the final line at parent approval.
