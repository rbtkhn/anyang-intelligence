# Learning Core Membrane Notes

These notes describe how Learning Core should transfer lessons while protecting child, parent, and household context.

## Transfer Boundary

Allowed transfer:

- generic parent-review patterns
- child-safety review questions
- readiness and hold-gate discipline
- evidence-versus-mastery guardrails

Blocked transfer without explicit parent approval:

- child-specific observations
- private household context
- saved or shared learner information
- family-specific support concerns or child-facing language

## Private Boundary

Private or lane-local material includes:

- learner-specific details
- family schedule and household constraints
- parent notes and approval boundaries
- support concerns, private reflections, and portfolio evidence

## Working Rule

Learning Core may transfer generic learning-governance patterns.

It may not transfer private child or household context, or any unapproved child-facing language, without explicit parent approval.
