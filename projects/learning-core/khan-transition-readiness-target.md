# Khan Transition Readiness Target

This document defines the **transition-readiness threshold** between **Khan Academy Kids** and the main **Khan Academy** experience as a deliberate developmental target for Learning Core.

The point is not to say:

- the learner finished Khan Kids
- the learner proved mastery
- the learner has reached a grade verdict

The point is to say:

- the learner is becoming ready for more structured, more self-propelled digital learning

That readiness threshold can be used as a meaningful target that Learning Core pushes toward deliberately.

## Core Idea

The Khan transition threshold is not just a handoff rule.

It is a **target capability bundle**.

Learning Core can use that bundle as a practical aim because it develops abilities that matter beyond Khan itself:

- reading stamina
- instruction-following
- digital independence
- frustration recovery
- sustained focus
- structured follow-through

These help with:

- digital lessons generally
- workbook use
- guided reading
- more independent practice
- parent ease

## The Readiness Target

Learning Core should treat the target as:

`readiness for structured self-propelled digital learning`

That is the developmental objective.

The learner does not need perfection.

The learner does need enough visible strength across several capabilities that the next environment can actually help rather than merely frustrate.

## Capability Bundle

The transition-readiness target should include these capabilities.

### 1. Instruction Reading Capacity

The learner is becoming able to:

- read enough interface language to participate
- read simple instructions without everything being mediated
- tolerate more text than the earliest starter flow

Why it matters:

- the main Khan environment requires more reading participation

What Learning Core can deliberately build:

- daily reading practice
- short instruction-following tasks
- parent read-then-release routines
- reading basket choices that build confidence and stamina

### 2. Sustained Attention Capacity

The learner is becoming able to:

- stay with a task beyond the novelty burst
- finish a meaningful chunk
- tolerate a more structured sequence

Why it matters:

- main Khan works better when the learner can stay in the lane long enough to benefit from it

What Learning Core can deliberately build:

- short but steady practice blocks
- simple finish-what-we-start expectations
- gradual time extension
- observation of fatigue and timing fit

### 3. Digital Navigation Capacity

The learner is becoming able to:

- move through a digital learning flow with less rescue
- choose among options without immediate confusion
- recover from small mistakes

Why it matters:

- the main Khan environment asks for more interface handling

What Learning Core can deliberately build:

- light supervised app independence
- short "you try first" moments
- repeated familiar digital routines
- calm error recovery language

### 4. Correction Tolerance Capacity

The learner is becoming able to:

- experience a wrong answer without shutdown
- try again after feedback
- accept mild correction pressure

Why it matters:

- structured digital learning becomes more useful when correction does not end the session

What Learning Core can deliberately build:

- normalize mistakes
- use low-stakes practice
- praise recovery, not only correctness
- shorten sessions before overload becomes defeat

### 5. Structured Follow-Through Capacity

The learner is becoming able to:

- continue through a short sequence
- return to a lane more than once
- tolerate repetition in service of learning

Why it matters:

- main Khan is stronger when the learner can participate in a sequence rather than only sampling novelty

What Learning Core can deliberately build:

- repeatable weekly rhythm
- familiar activity lanes
- small sequence completion
- parent language that frames continuation as normal

### 6. Parent-Fit Capacity

The household is becoming able to:

- support the required supervision
- place the block in a stable rhythm
- keep the transition from becoming a struggle

Why it matters:

- readiness is not just in the child; it is in the operating environment

What Learning Core can deliberately build:

- realistic time blocks
- clear screen-time rules
- lower-prep support structures
- parent observation confidence

## What Counts As Approaching The Threshold

The learner is **approaching threshold** when:

- willingness is improving
- attention is stretching
- prompt dependence is decreasing
- reading tolerance is improving
- frustration is more recoverable
- structured tasks are becoming more doable

At this stage, Learning Core should still focus on building capacity rather than forcing the transition.

## What Counts As At The Threshold

The learner is **at threshold** when several of the following are visible together:

- the learner can participate in a more structured digital task without constant rescue
- the learner can read enough interface or lesson text to stay in the flow
- the learner can continue after mistakes with manageable support
- the learner can stay with a short structured sequence
- the parent believes a limited trial in main Khan Academy is realistic

This should trigger a **trial transition**, not a permanent promotion.

## What Course Completion Means Here

Course completion in Khan Academy Kids may support the readiness picture if it appears alongside the capability bundle.

It may suggest:

- confidence
- persistence
- familiarity with the current environment

It does not establish:

- mastery
- grade placement
- automatic readiness

Completion is supporting evidence, not the threshold itself.

## How Learning Core Should Push Toward The Target

Learning Core should deliberately shape the month toward this threshold by increasing:

- reading stamina
- follow-through
- digital independence
- correction tolerance
- structured practice tolerance

It should do so gradually, with parent approval, and without turning the month into a hidden test-prep regime.

## Example Push Moves

- Keep Khan Academy Kids short but intentional, with parent observation.
- Add reading-basket selections that strengthen confidence and instruction tolerance.
- Use simple hands-on or workbook tasks that require finishing a small sequence.
- Build "try first, then help" routines.
- Celebrate recovery and continuation, not just right answers.
- Move from purely playful flows toward slightly more structured ones when the learner is ready.

## Governance Boundary

This readiness target may be used to guide planning.

It may not be used to:

- declare mastery
- assign grade placement
- override parent authority
- imply that a digital threshold equals the whole learner

## Success Criteria

This readiness target is working when:

- Learning Core has a concrete capability aim to build toward
- the learner becomes more able to handle structured digital learning
- the parent can see why specific routines are being recommended
- the eventual Khan transition feels earned and usable
- the threshold functions as a developmental target, not a grading system
