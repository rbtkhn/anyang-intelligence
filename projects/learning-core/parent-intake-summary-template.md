# Parent-Verified Intake Summary Template

This reusable blank template turns a guided parent intake into a concise readiness packet. A completed real-family copy is tenant-private and must remain outside Git.

Do not fill this repository template with real learner, parent, household, contact, school, support, or evidence details.
