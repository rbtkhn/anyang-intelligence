Usable claims:
- Open-ended agent value may depend heavily on problem selection quality, not only execution quality.
- Bounded workspace access can be a first-class part of the prompt, not an afterthought.
- A system may be more valuable when it returns a reviewed problem definition plus artifact than when it simply executes a requested task.
- Different agent stacks may produce complementary judgments, making comparative runs useful when the task is ambiguous.
- Harness ergonomics may shape real adoption as much as raw strategic intelligence.
