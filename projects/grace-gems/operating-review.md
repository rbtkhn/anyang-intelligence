# Grace Gems Operating Review

This is the recurring operating review surface for Grace Gems.

Cadence: weekly, plus event-driven review before major listing publish/promotion decisions.

## Review Questions

### Listing Health

- Which listings are ready, blocked, underperforming, or at policy risk?
- Which listing has the highest upside if improved this week?
- Which products need stronger first-image, search, or trust clarity?

### Margin and Promotion

- Which listings should not receive ad spend?
- Where are margin assumptions weakest?
- Which promotion candidates are strong enough to justify visibility effort?

### Customer Signals

- Which buyer questions repeated this week?
- What confusion patterns point to preventable listing problems?
- Which support responses should become listing or FAQ updates?

### Storefront Fit

- Which products still fit their current storefront?
- Where is store-specific positioning creating trust or confusion?

### Next Actions

- What should be published, revised, held, retired, or promoted next?

### Turn Usefulness Review

- Which turn created the clearest owner-ready next action this cycle?
- Which turn looked complete but still hid cleanup in listing facts, trust language, or promotion readiness?
- Where did the system stop too early and leave the owner to reconstruct the decision?
- Where did any recommendation get too close to changing promises, pricing, or publication state without approval?
- Which repeated failure mode should become a tighter owner-review receipt, checklist, or listing gate?
