# Customers

Customer installations are concrete operating contexts where Anyang Intelligence acts as the Executive Operating System.

The goal is to prove the pattern across different kinds of complexity before turning repeated workflows into software.

## Current Installations

| Installation | Domain | Executive OS proof |
| --- | --- | --- |
| [Grace Gems](grace-gems/README.md) | Marketplace business | Listing gates, margin discipline, customer-promise clarity, weekly marketplace execution |
| [Mountain Home](mountain-home/README.md) | Property and asset governance | Risk register, continuity planning, seasonal readiness, maintenance prioritization |

## Portfolio Logic

The early portfolio should stay small and sharp.

Grace Gems shows that an Executive OS can run commercial operating loops. Mountain Home shows that the same pattern can govern a real-world asset where risk, readiness, and continuity matter more than revenue.

Together they test the core Anyang claim: an Executive OS is not a chatbot or a dashboard. It is a living executive layer that keeps a complex operating context coherent.
