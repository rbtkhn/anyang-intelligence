# Continuity Invitation Template

This template is the parent-facing invitation to continue after the free 30-day plan.

Its job is to make the transition into the monthly continuity subscription feel earned, clear, and helpful.

It should only be used after the first month produced real parent-visible value and a visible next-step need.

## Purpose

Use this template when:

- the first 30-day plan has been used
- month-one observation produced meaningful patterns
- the parent now has a clearer picture of the child, household, and next-step questions
- the continuity layer can honestly help with month two

Do not use this template as a default upsell message before the first month has generated real evidence.

## Invitation Logic

The invitation should follow this sequence:

```text
month-one evidence
  -> parent-visible value
  -> natural next need
  -> continuity offer
  -> parent choice
```

If the message skips the first three steps and jumps straight to features, it will feel salesy.

## Parent-Facing Template

Use or adapt:

Hi,

Thank you for working through this first month.

Here are a few patterns that became clearer:

- `[pattern 1 from the month]`
- `[pattern 2 from the month]`
- `[pattern 3 from the month, if useful]`

What this month seems to have helped make clearer:

- `[what the parent can now see or decide more clearly]`
- `[what became easier to notice, support, or adjust]`

A natural next question for month two is:

- `[next-month question or decision that now matters]`

If you want, the next layer is the monthly continuity subscription.

That continuity layer is for families who want ongoing help with:

- interpreting new observations
- refreshing the reading basket, rhythm, or next-step plan
- turning portfolio and school/home signals into clearer parent decisions
- staying in a steady support loop without restarting from zero

The continuity subscription currently includes:

- chat-group participation
- premium newsletters
- personalized ongoing consultation
- early access to new upgrades and features

You do not need to continue for the first month to have been worthwhile. The goal is simply to offer a useful next layer if you want ongoing depth and continuity.

If you want to continue, the next step would be:

- `[month-two starting move]`

If you would rather use the current plan independently for now, that is completely fine too.

## Short Version

Use this when the parent needs a lighter message:

```text
This first month helped make a few things clearer:

- [pattern]
- [pattern]

If you want, month two can be a continuity month instead of a restart month.

That ongoing layer is for interpreting new observations, refreshing next steps, and keeping the system alive without rebuilding everything from zero.
```

## Evidence Prompts For The Operator

Before sending the invitation, fill in:

- What did the child choose, avoid, repeat, or ask for?
- What did the reading basket reveal?
- What did school and home together reveal?
- What did the portfolio make visible?
- What decision is now easier than it was 30 days ago?
- What next decision is now pressing enough that continuity would help?

If those answers are weak, do not send the invitation yet.

## Good Evidence Examples

Good invitation evidence sounds like:

- The child kept returning to animal nonfiction and oral explanation, but school writing tasks created visible friction.
- The parent can now see that a lighter weekday rhythm and a stronger weekend reading basket produce better engagement.
- The portfolio shows strong curiosity and verbal reasoning, but thin writing evidence, so the next question is how to support writing without increasing resistance.

These examples point to real continuity value.

## Bad Invitation Signals

Do not send the invitation in a way that says or implies:

- the free plan was only an introduction
- more support is needed because the first month was insufficient
- the parent must subscribe to stay safe, supported, or responsible
- features matter more than the child or household patterns

The invitation should arise from insight, not pressure.

## Month-Two Starting Moves

Useful month-two starting moves include:

- refresh the reading basket from observed choices
- simplify the weekday rhythm around school energy realities
- build a deeper learner profile from month-one evidence
- focus the next month on one visible friction point
- strengthen the portfolio capture rhythm without adding paperwork

The invitation should name one likely starting move instead of sounding abstract.

## Boundary

This template is not:

- a pressure sequence
- a scarcity message
- a diagnosis or support warning
- a substitute for parent judgment

It is a gentle continuity invitation built from real month-one evidence.
