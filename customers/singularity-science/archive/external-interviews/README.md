# External Interviews Archive

This folder stores internal analysis material for public interview transcripts and source notes that are relevant to Singularity Science but do not belong to a dedicated recurring source lane such as Innermost Loop, Moonshots, or Nate B. Jones.

External Interviews use the same archive discipline as the rest of Singularity Science: preserve source material internally, extract seams, verify unstable claims, and route only membrane-safe primitives or questions outward.

```text
external-interviews/
  transcripts/
  source-notes/
  analyses/
  source-note-template.md
  analysis-template.md
  roi-ledger.md
```

Use these artifacts:

- [source-note-template.md](source-note-template.md)
- [analysis-template.md](analysis-template.md)
- [roi-ledger.md](roi-ledger.md)

## External Interview ROI Pipeline

External interviews are optimized for **framework yield per contested claim**, not for treating an interviewee's worldview as Anyang Intelligence doctrine.

Standard flow:

1. Land transcript inside this archive lane.
2. Create a source note with speaker, platform, dominant seams, rhetoric risk, verification-priority claims, and rights concerns.
3. Create seam-first analysis that extracts 2-5 seams and assigns exactly one ROI disposition to each seam.
4. Update [roi-ledger.md](roi-ledger.md) with one row per seam.
5. Preserve, verify, route, or hold based on the seam disposition.

ROI dispositions:

- `primitive-candidate`
- `needs-verification`
- `lane-test-ready`
- `worldview-only`
- `preserved`

The standard routing packet for `lane-test-ready` seams is:

```text
Source interview:
Seam:
Transferable question or checklist:
Receiving lane:
Membrane classification:
Human authority required:
Evidence still needed:
What stays inside Singularity Science:
```

## Boundary

Full transcripts are internal analysis material. Do not quote, republish, or customer-route transcript text without rights review and source attribution.

External interview claims should not become Anyang Intelligence doctrine merely because the speaker is prominent, persuasive, or technically credible.
