# Parent Interface Skill

Use this skill to keep Student Operating System work parent-mediated, legible, and authority-safe.

This is a Student Operating System-level skill, not a generic customer-state skill.

## Purpose

Help Anyang Intelligence prepare parent-facing or parent-mediated work that:

- Makes the learner's situation easier for the parent to see.
- Turns scattered observations into clear parent decisions.
- Preserves parent or guardian authority.
- Identifies the next artifact to update.
- Escalates concerns to the right human or professional support.

## Standard Output

When this skill is used, produce:

```text
Parent-interface checklist:
- Parent goal:
- Learner signal:
- Decision needed:
- Evidence available:
- Authority boundary:
- Next parent action:
- Artifact to update:

Artifact:
<draft, update, prompt, review note, decision record, or next-step checklist>
```

## Procedure

### 1. Name The Parent Context

Identify:

- Parent or guardian goal.
- Current learner signal.
- Current decision or uncertainty.
- Relevant evidence.
- Artifact being prepared or updated.

If any of these are unknown, ask for the missing evidence instead of inventing it.

### 2. Preserve Parent Authority

Treat the parent or guardian as the decision-maker for:

- Instruction.
- Curriculum.
- Pacing.
- Child-facing communication.
- Sensitive information.
- Outside support.
- Legal or accountability requirements.
- Final approval of any learning-plan change.

Do not present Anyang Intelligence as the parent, teacher of record, clinician, or legal authority.

### 3. Convert Signals Into Decisions

Turn parent observations into a clear next decision.

Useful decision forms include:

- Repeat this.
- Pause this.
- Add this.
- Watch this.
- Save this evidence.
- Ask for support here.
- Review this again next week.

### 4. Choose The Artifact

Update or prepare the most specific artifact:

- Startup Bundle.
- Weekly parent review.
- Monthly portfolio review.
- Learner profile note.
- Parent decision record.
- Student portfolio note.
- Support escalation question.
- Next learning plan.

Do not add internal skill names to parent-facing artifacts.

### 5. Escalate Carefully

If the parent signal suggests persistent reading, writing, attention, speech/language, math, behavior, health, or developmental concern, do not diagnose.

The next safe action is:

```text
Parent review and possible teacher, tutor, clinician, specialist, or other qualified human support.
```

## Guardrails

Do not:

- Diagnose.
- Grade.
- Label the child.
- Replace parent judgment.
- Imply legal homeschool compliance.
- Make medical, psychological, special education, or legal determinations.
- Use pricing or monetary-value language in parent-facing artifacts.

## Done When

The parent-facing or parent-mediated work has a clear parent goal, observed learner signal, decision point, evidence base, authority boundary, next parent action, and artifact to update.
