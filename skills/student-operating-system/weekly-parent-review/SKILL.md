# Weekly Parent Review Skill

Use this skill to turn a week of scattered learning moments into a clear parent-reviewed next step.

This is a Student Operating System-level skill, not a generic weekly report.

## Purpose

Help Anyang Intelligence produce weekly reviews that:

- Make the week easier for the parent to understand.
- Identify what to repeat, pause, add, or watch.
- Preserve useful evidence.
- Update the learner profile when appropriate.
- Prepare the next learning plan.

## Standard Output

When this skill is used, produce:

```text
Weekly-parent-review checklist:
- Week covered:
- Parent observations:
- Student signals:
- Evidence saved:
- What worked:
- What dragged:
- Decision for next week:
- Artifact to update:

Artifact:
<weekly parent review, next-week decision note, learner profile update, portfolio note, or learning-plan input>
```

## Procedure

### 1. Summarize The Week

Use parent-reviewed evidence from:

- Daily rhythm notes.
- Student survey or reflection.
- Reading basket use.
- Khan Academy Kids observations.
- Hands-on math or writing work.
- Student portfolio artifacts.
- Proud work.
- Parent questions.

Do not require perfect records. A useful weekly review can be short.

### 2. Extract Decisions

End with practical parent decisions:

- Repeat this.
- Pause this.
- Add this.
- Watch this.
- Save this.
- Ask for support here.
- Try this next week.

### 3. Update The Right Artifact

Choose the most specific artifact:

- Weekly parent review.
- Learner profile.
- Student portfolio.
- Reading basket.
- Monthly portfolio review input.
- Next learning plan.
- Parent decision record.

Do not add internal skill names to parent-facing artifacts.

### 4. Route Concerns Carefully

If evidence suggests persistent reading, writing, attention, speech/language, math, behavior, health, or developmental concern, do not diagnose.

The next safe action is parent review and possible teacher, tutor, clinician, specialist, or other qualified human support.

## Guardrails

Do not:

- Grade the week.
- Blame the child or parent.
- Diagnose.
- Treat app progress as proof of mastery.
- Replace parent judgment.
- Imply legal homeschool compliance.
- Use pricing or monetary-value language in parent-facing artifacts.

## Done When

The weekly review names what happened, what it suggests, what evidence to preserve, what should change next week, and which artifact should be updated.
