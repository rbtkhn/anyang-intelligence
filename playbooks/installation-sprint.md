# Installation Sprint

## Purpose

Install a first working Executive OS for an organization in four to six weeks.

## Week 1: Business Intake

- Map the organization, leadership team, products, customers, and operating cadence.
- Identify where business truth currently lives.
- Select the first executive pain point.

## Week 2: Executive Memory

- Build the company context map.
- Define memory objects.
- Capture initial goals, projects, risks, decisions, and metrics.

## Week 3: Decision Workflow

- Install the decision memo template.
- Create the decision log.
- Run one real decision through the system.

## Week 4: Coordination Loop

- Install the weekly executive review.
- Connect projects, owners, blockers, and follow-ups.
- Run the first operating review.

## Week 5: Proof of Value

- Identify the first measurable coordination win.
- Compare before and after.
- Update the risk register.

## Week 6: Handoff

- Document the operating cadence.
- Confirm governance rules.
- Review what should be automated or productized next.
